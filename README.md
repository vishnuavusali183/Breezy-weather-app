# 🌬️ Breezy Weather App

A beautiful, responsive weather app built with HTML, CSS, and JavaScript. Get real-time weather conditions and a 5-day forecast for any city in the world.

![Breezy Weather App](https://img.shields.io/badge/Weather-App-blue?style=for-the-badge&logo=cloud&logoColor=white)
![No API Key](https://img.shields.io/badge/API%20Key-Not%20Required-green?style=for-the-badge)
![Open Source](https://img.shields.io/badge/Open-Source-orange?style=for-the-badge)

---

## ✨ Features

- 🔍 Search any city worldwide
- 🌡️ Current temperature with feels-like
- 💧 Humidity, wind speed, visibility, pressure
- 🌅 Live sunrise & sunset arc
- 📅 5-day weather forecast
- 🌧️ Animated rain & snow effects
- 🎨 Dynamic sky background based on weather
- °C / °F toggle
- 📱 Fully responsive on mobile and desktop

## 📦 How to Run Locally

1. **Download** the `weather-app.html` file
2. **Open** it in Chrome or Edge
3. **Search** for any city and enjoy! 🌤️

> ⚠️ Use Chrome or Edge for best results. Firefox may block API requests from local files.

## 🌐 API Used

This app uses the **[Open-Meteo API](https://open-meteo.com/)** — a free, open-source weather API with no API key required.

- Geocoding: `https://geocoding-api.open-meteo.com/v1/search`
- Forecast: `https://api.open-meteo.com/v1/forecast`

## 📸 Weather Conditions Supported

| Condition | Background | Effect |
|-----------|-----------|--------|
| Clear sky | Deep blue gradient | — |
| Cloudy | Dark blue-grey | — |
| Rain | Dark stormy | 🌧️ Animated rain |
| Snow | Blue-grey | ❄️ Animated snow |
| Thunderstorm | Near black | 🌧️ Heavy rain |
